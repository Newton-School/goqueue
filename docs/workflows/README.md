# Workflows

Phase 7 workflow documentation is split by concern:

- `canvas.md` covers public signature, chain, group, and chord usage.
- `redis-state.md` covers Redis keys, scripts, and idempotency behavior.
- `failure-semantics.md` covers chain, group, and chord terminal behavior.
- `metadata.md` covers reserved workflow metadata keys.
- `worker-advancement.md` covers worker ordering and acknowledgement safety.
- `testing.md` covers recommended package-level verification.
